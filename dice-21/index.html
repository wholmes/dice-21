Dice 21 needs a local server

Browsers block file:// from loading ES modules (the game scripts). Run the dev server, then open the app over HTTP.

From the repo folder (where package.json is):

npm run dev

Then visit http://localhost:5173/ (Dice 21) or /dice-21/

After a production build: npm run preview and use the same path.

Unlocking higher bet sizes

Bigger chips unlock over your whole career, not a single session. You only need to satisfy one unlock path (standard or a “hot” shortcut)—whichever you hit first.

Two stats

  • Hands — lifetime hands finished (the first number in pairs like 96/15).
  • $ won — lifetime total of pot payouts on your wins (full pot each time, not net profit). Shown like $82/$250: you have $82 toward a $250 requirement.

Example: unlocking $5 max bet

  • Standard — at least 15 lifetime hands and at least $250 in lifetime pot wins.
  • Hot A — 10 hands and $320 won (fewer hands, more winnings).
  • Hot B — 25 hands and $200 won (more hands, lower $ bar).

Higher tiers

The same pattern continues for $25 and $100 — standard thresholds plus two hot shortcuts each time.
